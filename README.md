# GitOps Microservices Deployment

A production-inspired GitOps project demonstrating deployment and management of containerized microservices on Kubernetes using Docker, Helm, ArgoCD, and GitHub Actions.

## Business Domain

Smart Logistics Platform

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Docker
- Kubernetes
- Helm
- ArgoCD
- GitHub Actions
- Prometheus
- Grafana
- Loki

## Project Roadmap

- Week 1: Foundation & Microservices
- Week 2: Containerization & Kubernetes
- Week 3: Helm & CI Pipeline
- Week 4: GitOps with ArgoCD
- Week 5: Observability, Scaling & Security
- Week 6: Production Readiness & Portfolio
